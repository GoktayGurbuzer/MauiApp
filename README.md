# .NET MAUI

.NET Multi-platform App UI (.NET MAUI), tek bir kod tabanında C# ve XAML kullanarak modern, çok platformlu, yerel olarak derlenmiş iOS, Android, macOS ve Windows uygulamaları oluşturmaya yönelik bir çerçevedir.

Bu repo'da [dotnet][1] youtube sayfasında bulunan ".NET MAUI for Beginners" eğitim videosunun kodlara dökülmüş halini bulabilirsiniz.

## Eğitim İçeriğinde Neler Var?
- Masaüstü uygulama oluşturma
- Kullanıcı arayüzü oluşturma
- URI yapısı ile sayfalar arası geçiş
- Cihaz özelliklerine ulaşmak
